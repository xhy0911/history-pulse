# 史海新萃 · History Pulse

## 手机使用指南

### 方法一：GitHub Actions（全自动，推荐）

这套方案已经设置好了，部署完成后：
1. 每天 UTC 0:00（北京时间 8:00）自动运行
2. 自动抓取 RSS + 生成网站 + 部署到 GitHub Pages
3. 自动推送消息到你的微信

你只需要在手机上：
- 打开 https://xhy0911.github.io/history-pulse/ 浏览
- 在微信里接收每日推送

### 方法二：手动触发（手机上也可以）

1. 打开 https://github.com/xhy0911/history-pulse
2. 点 Actions 标签
3. 点左侧 "Daily Update"
4. 点 "Run workflow" → 绿色按钮

### 查看网站

手机浏览器打开：
https://xhy0911.github.io/history-pulse/
